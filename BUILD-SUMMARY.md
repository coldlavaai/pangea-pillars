## 🔧 Maintenance & Updates

### Easy Content Updates
To update content on any page:
1. Edit the page file in `app/[page-name]/page.tsx`
2. Update the `sections` or `faqs` arrays
3. Commit and push to GitHub
4. Vercel auto-deploys within 2 minutes

### Adding New Pages
The `PillarPage` component is reusable:
```tsx
<PillarPage
  title="Your Title"
  intro="Your intro"
  sections={[...]}
  faqs={[...]}
  schemaData={{...}}
  relatedPages={[...]}
/>
```

### Schema Markup Updates
Edit `components/SchemaMarkup.tsx` to modify structured data.
